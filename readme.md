
# Array di Squadre di Calcio

## Esercizio

Creare un array di oggetti di squadre di calcio.

### Requisiti

- Ogni squadra avrà le proprietà: `nome`, `puntiFatti`, `falliSubiti`
- `nome` sarà compilato manualmente, le altre proprietà inizialmente a 0
- Generare numeri random per `puntiFatti` e `falliSubiti`
- Creare un nuovo array contenente solo `nome` e `falliSubiti`
